# DSA-Student-Record-Manager
This project is a console-based "Student Record Management System" implemented in C++ using a "Doubly Linked List (DLL)" custom data structure.

Core Features & Functionalities
1. Flexible Insertion
Insert a new student record at the start or end of the list.
Insert a new student record directly before or after a specific student's Registration Number.

2. Deletion
Delete a specific student record by entering their Registration Number.

3. Traversal & Display
Display all student records in forward order (from first to last).
Display all student records in reverse order (from last to first).

4. Linear Search
Search and fetch student details by Registration Number.
Search and fetch student details by Student Name.

5. Memory Management
Custom destructor implemented in DLinkedList to clear memory dynamically upon exit.
